Política de cambio


Condiciones aplicables para cambios o devoluciones de productos.

  • Si el cliente opta por RECOJO EN TIENDA, es su responsabilidad revisar el estado del producto antes de retirarse de tienda.

  • Solo aplicarán cambios para aquellos productos que hayan llegado incompletos o con alguna pieza faltante. De no tener el producto para cambio, el cliente podrá usar el monto para uno nuevo o solicitar el reembolso.

  • La devolución del dinero se realizará al mismo medio de pago con el que se realizó la compra.
